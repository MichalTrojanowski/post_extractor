# Post Extractor (ZTNBD 2017)

Wymagania:
- python3
- build-essential
- pliki wejściowe*

Instalacja:
```bash
make init
```

Uruchomienie:
```bash
make run
```


*Pliki wejściowe w formacie json należy umieścić w katalogu `data/in`. Znaleźć je można w katalogu `statement` danych przykładowych znajdujących się na chmurze google.
